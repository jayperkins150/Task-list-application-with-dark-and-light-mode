# Task List Application with Dark and Light Mode

A task management application that allows users to organise daily activities while switching between dark and light themes for a personalised experience.

## Features

- Create tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Toggle between dark and light mode
- Responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

### Clone the repository

```bash
git clone https://github.com/jayperkins150/Task-list-application-with-dark-and-light-mode.git
```

### Run locally

Open `index.html` in your browser.
